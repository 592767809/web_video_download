
# 基于tampermonkey的下载方法

## 一、安装环境
### 1.你须要浏览器中安装Tampermonkey插件
### 2.下载解压项目，并在Tampermonkey中加载main.js脚本
### 3.运行main.exe
### 4.你需要安装m3u8批量下载器，并在使用前打开下载软件，否则在拦截到视频地址后无法发送下载请求https://www.52pojie.cn/thread-1374045-1-1.html

## 二、优点
### 1.你无需考虑cookie的问题
### 2.无需逆向接口加密参数
### 3.可以使用js完成各种操作
### 4.配合python脚本可以完成多元化操作
